# Shubham Raj Sharma - Portfolio

My personal portfolio focused on **Software Development Engineering** and **Backend Development**.

Live Site: [View Portfolio](https://shubh100802.github.io/Portfolio/)

## Highlights

- Hero, About, Projects, Skills, Tools, Hobbies, and Contact sections
- Responsive UI for desktop and mobile
- Theme toggle (dark/light mode)
- AOS scroll animations
- Interactive project cards
- Contact form powered by EmailJS
- Social footer (GitHub, LinkedIn, Instagram, Email)
- Resume download support

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- AOS (Animate On Scroll)
- Font Awesome
- EmailJS
